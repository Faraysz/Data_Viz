## 📖 Tentang Project

Finance Dashboard adalah aplikasi web modern yang membantu pengguna mengelola keuangan pribadi dengan cara yang visual dan intuitif. Dibangun dengan teknologi terbaru, aplikasi ini menyediakan fitur tracking transaksi, visualisasi data interaktif, dan analisis keuangan yang mudah dipahami.

Project ini dikembangkan sebagai bagian dari portofolio untuk mendemonstrasikan kemampuan full-stack development dengan fokus pada:
- 🎨 UI/UX yang clean dan responsive
- 📊 Data visualization yang meaningful
- 🗄️ Database design yang efisien
- ⚡ Performance optimization

## ✨ Fitur Utama

### 📊 Dashboard Overview
- **Real-time Summary** - Lihat total saldo, pemasukan, dan pengeluaran dalam satu pandangan
- **Interactive Charts** - Grafik arus kas 6 bulan terakhir dengan Recharts
- **Category Breakdown** - Distribusi pengeluaran per kategori dengan pie chart
- **Recent Transactions** - List 5 transaksi terakhir dengan detail lengkap

### 💳 Transaction Management
- **CRUD Operations** - Tambah, edit, hapus transaksi dengan mudah
- **Multiple Categories** - Kategorisasi otomatis untuk pemasukan dan pengeluaran
- **Date Tracking** - Catat tanggal transaksi dengan date picker
- **Bulk Actions** - Hapus multiple transaksi sekaligus (coming soon)

### 📈 Analytics & Insights
- **Trend Analysis** - Lihat pattern pengeluaran dan pemasukan
- **Budget Tracking** - Set budget per kategori dan monitor usage (coming soon)
- **Export Data** - Download laporan dalam format CSV/PDF (coming soon)

### 🎨 User Experience
- **Responsive Design** - Akses dari desktop, tablet, atau mobile
- **Dark/Light Mode** - Toggle tema sesuai preferensi (coming soon)
- **Fast Performance** - Optimasi dengan Next.js App Router
- **Smooth Animations** - Transisi yang halus dan profesional

## 🛠️ Tech Stack

### Frontend
- **Framework:** [Next.js 14](https://nextjs.org/) (App Router)
- **Language:** [TypeScript](https://www.typescriptlang.org/)
- **Styling:** [Tailwind CSS](https://tailwindcss.com/)
- **Charts:** [Recharts](https://recharts.org/)
- **Icons:** [Lucide React](https://lucide.dev/)
- **Utilities:** clsx, tailwind-merge

### Backend & Database
- **ORM:** [Prisma](https://www.prisma.io/)
- **Database:** [PostgreSQL](https://www.postgresql.org/)
- **Hosting:** [Supabase](https://supabase.com/)

### Development Tools
- **Package Manager:** npm
- **Code Quality:** ESLint, Prettier
- **Deployment:** [Vercel](https://vercel.com/)

- [PostgreSQL](https://www.postgresql.org/) database (atau gunakan [Supabase](https://supabase.com/))
